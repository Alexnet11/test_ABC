# Система управления скидками на Python (OOP)

Этот проект демонстрирует использование **абстрактных классов** и **полиморфизма** в Python для создания гибкой системы расчета скидок в интернет-магазине.

## Особенности
* Использование модуля `abc` для создания абстрактной базы.
* Реализация различных типов скидок (процентная, фиксированная, комбинированная).
* Цветной вывод в консоль с помощью `colorama`.
* Методы с реализацией по умолчанию и переопределением логики.
